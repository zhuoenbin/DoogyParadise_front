# DoogyParadise寵物平台網站

這是一個關於寵物的多系統平台，包含會員、商城、活動、旅館、動態牆。 會員系統:登入、註冊、忘記密碼、Google OAuth2第三方登入。

## 這是網站的前端程式

使用Vue3框架進行開發，套件部份是使用Bootstrap及Element Plus

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
